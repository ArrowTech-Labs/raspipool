Open Devices & services

Sync copies the integration and blueprints into /config. A Home Assistant restart is required after the first sync (or after any integration update) for the new files to load.

Next steps

  1. Click Sync / Update files above.
  2. Click Restart Home Assistant (first install only, or after an update).
  3. Flash raspipool-esp32.yaml to your ESP32 using the ESPHome add-on.
  4. Open Settings → Devices & services → Add integration and search for Raspipool.
  5. Complete the configuration wizard; your pool dashboard will appear automatically.
